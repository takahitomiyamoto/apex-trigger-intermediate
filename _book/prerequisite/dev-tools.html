Developer Edition

まず、Salesforce の無料の開発環境を入手しましょう。これを使って Trailhead にログインします。 すでに持っている場合は、次のセクションへ。

  1. Google Chrome または Firefox を開く。（Internet Explorer は非推奨）
  2. https://developer.salesforce.com/signup へアクセスする。
  3. 必要な情報を入力してサインアップする。
  4. 件名「Salesforce へようこそ: アカウントを確認してください」のメールを開いて「アカウントを確認」をクリックする。
  5. 好きなパスワードと秘密の質問を設定して「パスワードを変更」をクリックする。
  6. 歯車アイコン > 「設定」をクリックする。
  7. 左のペインで、「会社の設定」 > 「組織情報」をクリックする。
  8. 「編集」をクリックする。
  9. 「言語のデフォルト値」を "英語" に変更して「保存」をクリックする。

  • Visual Studio Code の Salesforce 対応

進めているうちに Trailhead Playground を作成する手順が出てきますが、前の単元で作成済みのものを起動して進めてください。 まだ作成中の場合はしばらく待ってください。

Trailhead Playground のログインパスワードリセット

Trailhead サイトから Trailhead Playground を起動する際にはログインパスワードが不要です。 しかし、Trailhead サイト以外から Trailhead Playground へログインしたい場合には、ログイン ID とパスワードが必要です。

  1. Trailhead サイトから Trailhead Playground を起動する。
  2. 歯車アイコン > 「設定」をクリックする。
  3. 左のペインで、「ユーザ」 > 「ユーザ」をクリックする。
  4. 自分の氏名と同じユーザを探して、「ユーザ名」の値を手元にメモしておく。
  5. 自分の氏名と同じユーザのチェックボックスをチェックし、「パスワードをリセット」をクリックする。
  6. ポップアップ画面の「OK」をクリックする。
  7. 件名「Your Developer Edition パスワードのリセット完了」のメールを開いて、本文のリンクをクリックする。
  8. 件名「Salesforce で ID を確認」のメールに記載されている確認コードを入力して「検証」をクリックする。
  9. 好きなパスワードを設定して「パスワードを変更」をクリックする。パスワードは手元にメモしておく。
  10. 設定したパスワードでログインできるかどうか確かめる。
    • アストロのアイコン > 「ログアウト」をクリックする。
    • メモしたユーザ名とパスワードを入力して「ログイン」をクリックする。
    • 「携帯電話を登録」画面が表示された場合は、「電話を登録しません」をクリックする。

VS Code Extensions

さきほど VS Code の拡張機能の Salesforce Extension Pack をインストールしましたが、他にも開発に欠かせない拡張機能が豊富に存在します。 ぜひインストールしましょう。

No. 必須 名前 オススメポイント
1 Y Bracket Pair Colorizer 括弧のペアを色付きで把握できる
2 Y Codey Midnight Salesforce 開発にぴったりのダークテーマ
3 Y Prettier - Code formatter コードを綺麗に整形してくれる
4 Y Todo Tree TODO コメントを一覧化できる
5 Y Trailing Spaces 余分なスペースを削除してくれる
6 Y zenkaku 余分な全角スペースをハイライトしてくれる
7 Apex PMD Apex コードの静的解析を実行できる
8 Codey High Noon Salesforce 開発にぴったりのライトテーマ
9 GitLens — Git supercharged Git のコミットの履歴を追跡しやすくなる
10 Live HTML Previewer HTML をプレビューできる
11 Live Share リアルタイムにコードを共有して共同作業できる
12 Output Colorizer ログに色付けしてくれる

Node.js & Yarn

インストール

Prettier の一括実行など各種コマンドを実行するために、Node.js をインストールしましょう。 次の URL からインストーラーをダウンロードしてください。基本的に LTS バージョンで十分です。

  • Node.js
  • Yarn

インストーラーをダブルクリックし、デフォルトのままインストールを完了させてください。

確認

次のコマンドでインストールが正常に完了していることを確認しましょう。 バージョンが表示されれば OK です。 

node -v
npm -v
yarn -v

設定

いま VS Code で開いている「VSCodeQuickstart」プロジェクトのルートに「package.json」ファイルを新規作成してください。

package.json

パッケージのバージョンアップ

以下のコマンドを実行して、パッケージのバージョンを最新化しましょう。 

yarn install
yarn gulp:init
