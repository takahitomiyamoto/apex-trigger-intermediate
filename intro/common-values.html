Common Values

Common Values

本チュートリアルが全体を通して大切にしている価値観です。

1. 習うより慣れよ

理解してから書くのではなく、書いたあとで理解しましょう。カンペキ主義は得策ではありません。

2. 写経せよ

お手本を見ながら書き写すことが上達の近道です。コピペは禁止。

3. ログと戯れよ

エラーが発生しても焦らないでください。落ち着いてログを 1 行ずつ読みましょう。
