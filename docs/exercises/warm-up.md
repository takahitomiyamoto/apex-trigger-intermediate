# Warm-up

## シンプルな Apex トリガを書いてみよう

まずは、シンプルな Apex トリガを書いてみましょう。

## ビジネス要件

これから取引先オブジェクトにサンプルデータを何件か作成しようと考えています。
ひと目でサンプルデータであることがわかるように、取引先名の先頭に `[サンプル]` という文字列を自動的に付与してください。

## アーキテクチャ要件

Apex トリガのみで実装してください。
Apex クラスなど他の資源を使用してはいけません。

## システム要件

| タイトル         | 内容             |
| :--------------- | :--------------- |
| Apex トリガ名    | `AccountTrigger` |
| 対象オブジェクト | 取引先           |
| 起動条件         | 新規作成時       |
